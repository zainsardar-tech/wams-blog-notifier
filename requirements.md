# Requirements - WAMS Blog Notifier

Ensure your system meets these requirements for the best experience.

## 🛠️ System Requirements

- **PHP**: 7.4 or higher.
- **WordPress**: 5.8 or higher.
- **MySQL**: 5.7 or higher.

## 🔑 API Credentials

- **Aztify API Key**: Required for sending WhatsApp notifications.
- **Aztify Sender Number**: A valid sender number registered on the Aztify platform.

## 🌐 Connectivity

- Your server must be able to make outbound HTTPS requests to `https://wams.aztify.com`.
- **cURL Extension**: PHP cURL must be enabled for API communication.
