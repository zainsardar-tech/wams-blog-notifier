# WAMS Blog Notifier

Automatically notify subscribers via WhatsApp when a new blog post is published on your WordPress site.

## 📖 Description

**WAMS Blog Notifier** is a lightweight, efficient WordPress plugin that bridges the gap between your blog and your audience. By integrating with the Aztify WAMS API, it ensures that your subscribers are always updated with your latest content, delivered directly to their WhatsApp.

## 🚀 Key Features

- **Automatic Real-time Notifications**
- **Customizable Message Templates**
- **Elegant Subscription Form**
- **Full Subscriber Management**
- **Detailed Activity Logs**
- **Developer Friendly**

For a full list of features, see [features.md](file:///home/zainsardar/Desktop/Visual%20Repo/SaaS%20Products/Wordpress%20Plugins/dummy/wams-blog-notifier/features.md).

## 🛠️ Requirements

- PHP 7.4+
- WordPress 5.8+
- Aztify API Key & Sender Number

For detailed requirements, see [requirements.md](file:///home/zainsardar/Desktop/Visual%20Repo/SaaS%20Products/Wordpress%20Plugins/dummy/wams-blog-notifier/requirements.md).

## 📥 Installation & Setup

1. Upload and activate the plugin.
2. Configure your API key in **WAMS Notifier** -> **Settings**.
3. Use the `[wams_subscribe]` shortcode to display the subscription form.

For a step-by-step guide, see [install.md](file:///home/zainsardar/Desktop/Visual%20Repo/SaaS%20Products/Wordpress%20Plugins/dummy/wams-blog-notifier/install.md).
