# Changelog - WAMS Blog Notifier

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-06

### Initial Release
- Automatically notify subscribers via WhatsApp when a new blog post is published.
- Customizable message templates with placeholders.
- Subscription form shortcode `[wams_subscribe]`.
- Admin settings for API key, sender number, and form customization.
- Subscriber management (add, edit, delete).
- Activity logging and API connection testing.
- Documentation and developer-friendly files added.
