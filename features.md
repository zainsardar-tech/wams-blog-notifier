# Features - WAMS Blog Notifier

The WAMS Blog Notifier is designed to seamlessly integrate WordPress with WhatsApp, allowing for real-time notifications to your subscribers.

## 🚀 Key Features

- **Automatic Notifications**: Instantly send a WhatsApp message to all subscribers whenever a new blog post is published.
- **Customizable Message Templates**: Personalize your notifications using placeholders like `{blog_name}`, `{post_title}`, and `{post_url}`.
- **Subscription Form**: Easily add a beautiful, WhatsApp-branded subscription form to any page or post using the `[wams_subscribe]` shortcode.
- **Admin Dashboard**:
    - **Subscriber Management**: View, add, edit, or delete subscribers directly from the WordPress admin.
    - **API Configuration**: Simple setup for Aztify API credentials.
    - **Customization**: Change form colors, titles, descriptions, and footer text to match your site's branding.
- **Connection Testing**: Built-in tool to verify your API connection and ensure everything is working correctly.
- **Activity Logging**: Keep track of sent notifications and errors for easy debugging.
- **Developer Friendly**: Clean, modular code structure for easy extension.
